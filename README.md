# IADL
### Information Assisted Dictionary Learning

The current repository contains the main code of the IADL algorithm and most of its auxiliary functions. It also includes the main folder of the Synthetic Dataset.

**Contents**

    IADL -> Folder with the source code + auxiliary functions
     ├─ IADL.m          -> Main code
     ├─ DuWProjectOpt.m -> Weighted l1-norm projection
     ├─ icaML.m         -> Initialization (infomax)
     └─ jadeR.m         -> Initilaization (JADE)
     
     Synth_Dataset  -> Forder tih the synthetic dataset
      └─ DataSet_Info.pdf -> Summary of the dataset
